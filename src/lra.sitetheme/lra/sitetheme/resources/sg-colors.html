Standardfarben

Grauwerte

Graustufen werden im Hinblick auf die Barrierefreiheit mit möglichst hohem Kontrast auf Basis eines 100%igen Schwarzwertes automatisch generiert. Da bei der Darstellung auf Monitoren ein reines Schwarz als zu kontrastreich angesehen wird, ist ein dunkler Grauwert $gray-dark als Standard für Fliesstext in Verwendung.

{% highlight scss %} $gray-base: #000 !default; $gray-darker: lighten($gray-base, 13.5%) !default; // #222 $gray-dark: lighten($gray-base, 20%) !default; // #333 $gray: lighten($gray-base, 33.5%) !default; // #555 $gray-light: lighten($gray-base, 46.7%) !default; // #777 $gray-lighter: lighten($gray-base, 93.5%) !default; // #eee {% endhighlight %}
